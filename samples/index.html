Medea Samples

  • SimpleInit - framework initialization, basic setup
  • MultipleViewports - shows how to initialize and populate multiple viewports
  • RotatingCube - a simple cube that rotates and changes color over time
  • RotatingTexturedCube - the same cube, this time with a texture image applied to it
  • FPSCamera - a simple labyrinth scene that can be traversed using standard FPS controls
  • FrustumCulling - demonstrates how automatic frustum culling greatly reduces the batch count
  • ModelLoading - load models from a JSON-based format (via assimp2json)
  • Skybox - Based on the ModelLoading sample, but renders a skybox background. Additionally, this samples uses FetchMod() to load its contents asynchronously
